# e-commerce-back-end

A backend for an eCommerce site

## Installation
```
git clone git@github.com:glowingmanagement/e-commerce-back-end.git
cd e-commerce-back-end
npm install
```

## Usage
1. Create env file based on the example provided
2. log into mysql by using:
```
mysql -u root -p
```
3. 
```
npm run seed
```
